# 介绍

关于 CatchAdmin Laravel 版，跟 TP 不同的是，Laravel 版本提供只提供了核心，你可以利用该核心开发任何后台应用。

# 环境要求

  • php7.3+ (需以下扩展)
    • Mbstring
    • Json
    • Openssl
    • Xml
    • Pdo
    • Zip
  • nginx
  • mysql
